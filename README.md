# Единая система профессионального развития ГГС Санкт-Петербурга

Прототип сквозного цикла: служащий → оператор → аналитика

## 🎯 Интерактивный прототип

**[Открыть прототип →](https://htmlpreview.github.io/?https://github.com/aminovvfbf-dev/ilya/blob/main/Прототип_система_профразвития_v2.html)**

### Три роли в системе:

1. **Служащий** — Личный кабинет
   - Моя образовательная траектория
   - Готовность к должностному росту
   - Отложенная оценка применения компетенций

2. **Оператор** — Паспорт и конструктор
   - Паспорт результатов программы ДПО (расчёт прироста компетенций)
   - Конструктор траектории (подбор по правилам: категория + дефициты)

3. **Руководитель Корпоративного университета** — Аналитика
   - Рейтинг программ по приросту компетенций
   - Разрыв «прошли обучение» → «применяют в работе»
   - Связь обучения и должностного роста

---

## 📋 Как использовать

1. Откройте интерактивный прототип по ссылке выше
2. Нажмите кнопки ролей для переключения между представлениями
3. В режиме оператора:
   - Выберите программу обучения
   - Заполните входные и итоговые измеры компетенций
   - Нажмите «Рассчитать результат» или «Заполнить демо-данными»
4. В режиме конструктора:
   - Выберите категорию должности
   - Отметьте выявленные дефициты
   - Нажмите «Подобрать траекторию»
5. Нажмите **▶ Демо-режим** для быстрого просмотра аналитического модуля

---

## 📊 Основные метрики

- **Прирост компетенций** — средняя разница итогового и входного замеров
- **Отложенная оценка** — подтверждение применения в работе через 3–6 месяцев
- **Разрыв прошли→применяют** — ключевой показатель эффективности программ
- **Готовность к росту** — показатель на основе подтверждённого прироста и применения
